# 简介
MyCommunity 是一个开放、包容、兼收并蓄的社团云平台。

# 如何使用

## 手动部署

前端
/client/my-community

``` shell
# 补全依赖，需要先安装yarn包管理工具
yarn install
# 启动服务
yarn serve
# 构建项目
yarn build
```

后台
/server/

``` shell
# 启动服务器，需要先安装go语言编译器
go run main.go
```
